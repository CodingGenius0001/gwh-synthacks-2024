Rules

Technology Usage
The participants can create their project using any technology they want including but not limited to: AI, ML, App development, IoT, Robotics etc.
Hardware Components
If hardware components are being used, the participants must procure the required equipment themselves.
No Paid APIs
Usage of any paid API is not permitted.
No Prewritten Code
Participants will not be allowed to use any prewritten code.
No Plagiarism
Plagiarism is not allowed in any case; it will cause the entire team to be disqualified.
Project Timing
Participants WILL NOT be allowed to work on their projects before the competition.
Rule Violations
If a team is found breaking any rule, they will have until the next auditing round to rectify their code.
